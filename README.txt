This program generates the rosters with the highest projected points for the Draft Kings Daily Fantasy. Currently NCAAB, NFL, NBA, and PGA are supported.
-Before use, the wd, source, and playerData destinations must be changed. 
Usage:
	modelOptimized(playerData, DESIREDSPORT(playerData), NUMBER_OF_LINEUPS_RETURNED)

DESIREDSPORT keys:
NCAAB -- collegeBB()
NFL -- NFL()
NBA -- NBA()
PGA -- golf()
